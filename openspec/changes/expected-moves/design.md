## Architecture

### Data Flow
```
IB Gateway (port 4002)
  → scripts/update-expected-moves-ib.py (cron 4:05 PM ET Mon-Fri)
    → data/expected-moves.json
      → expected-moves.html (EM dashboard)
      → watchlist.html openDetail() (EM banner in modal)
```

### File Structure
```
expected-moves.html          # New standalone EM page
watchlist.html               # Modified: EM banner in detail modal, table default
js/nav.js                    # Modified: added Expected Moves page link
data/expected-moves.json     # Data source (generated by scripts)
scripts/update-expected-moves-ib.py    # IB-powered EM calculator
scripts/update-expected-moves.py       # Polygon.io backup calculator
```

### Components

#### Expected Moves Page (`expected-moves.html`)
- Self-contained HTML with inline CSS/JS
- Loads `data/expected-moves.json` and `data/watchlist.json` on init
- Stats row: computed from loaded data (avg, max, min, counts)
- Tier tabs: re-renders table with selected tier's data
- Sortable columns: click header to sort asc/desc
- Risk meter: filled bar (`<div class="risk-fill">`) width = position %, color = risk color
- Alert tags: inline spans with pulse animation for actionable levels
- Row click: navigates to `watchlist.html?ticker={symbol}`

#### Watchlist EM Banner (`watchlist.html`)
- `loadExpectedMoves()` — async fetch of `data/expected-moves.json` into `emData`
- `buildExpectedMoveHTML(symbol, currentPrice)` — generates banner HTML
- Called in `openDetail()` before chart/info rendering
- URL param handler: reads `?ticker=` and auto-opens detail after watchlist loads

### Risk Calculation
```javascript
position = (currentPrice - lower) / (upper - lower) * 100
// Clamped 0-100 for visual bar, unclamped for labels (ABOVE/BELOW EM)
```

### Color Scale
```javascript
function getRiskColor(pct) {
  if (pct <= 20) return '#1b5e20';  // dark green
  if (pct <= 40) return '#4caf50';  // green
  if (pct <= 55) return '#8bc34a';  // light green
  if (pct <= 70) return '#ffeb3b';  // yellow
  if (pct <= 85) return '#ff9800';  // orange
  if (pct <= 100) return '#f44336'; // red
  return '#b71c1c';                  // dark red
}
```
