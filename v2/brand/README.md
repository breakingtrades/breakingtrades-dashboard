# BreakingTrades — Brand Assets

## Files

| File | What | Usage |
|------|------|-------|
| `logomark.svg` | Pulse mark only (square) | Favicon, app icon, social avatar, mobile nav |
| `wordmark.svg` | "BREAKING TRADES" with gradient pulse underline | Headers, banners, without the mark |
| `full-lockup.svg` | Mark + wordmark combined | Primary logo, marketing, desktop nav |
| `../favicon.svg` | Simplified mark for browser tab | Favicon (thicker strokes for 16px) |

## Brand Colors

| Color | Hex | Role |
|-------|-----|------|
| Cyan | `#00d4aa` | Primary brand, bullish, active |
| Gold | `#ffd700` | Pulse accent, breakout energy |
| Orange | `#ffa726` | Warning, approaching |
| Red | `#ef5350` | Bearish, exit, danger |
| Dark | `#0a0a12` | Primary background |
| Card | `#111122` | Card/surface background |
| Light | `#e0e0e8` | Primary text |
| Dim | `#8888aa` | Secondary text |

## Typography

- **Primary:** SF Mono → Fira Code → JetBrains Mono → monospace
- **Brand text:** BREAKING (700 weight, cyan) + TRADES (400 weight, dim gray)
- **Pulse underline:** Gradient from cyan (#00d4aa) → gold (#ffd700) → orange (#ffa726), spike at the break point between words

## Logo Concept

The **Market Pulse** — an EKG/heartbeat line forming a market chart. The market's vital signs: alive, volatile, breaking.

The gradient pulse underline beneath the wordmark spikes right at the junction between "BREAKING" and "TRADES" — the exact moment of the breakout. Gold dot marks the peak.

## Size Guidelines

- **Desktop nav:** Mark 24px + wordmark ~160px wide
- **Tablet nav:** Wordmark scaled to ~130px
- **Mobile nav:** Mark only (24px)
- **Social/marketing:** Full lockup SVG at any size
- **Favicon:** Simplified mark with thicker strokes
