## TradingView Embed Gotchas (IMPORTANT — READ THIS)
- **Studies array**: Use STRING IDs only! `'MASimple@tv-basicstudies'` — object syntax breaks silently
- **Can't override 2nd study instance**: `moving average_1`, `moving average#1` both fail — hard limitation
- **Valid range values**: `'1D'`, `'5D'`, `'1M'`, `'3M'`, `'6M'`, `'12M'`, `'60M'`, `'ALL'` — NOT `'1Y'`, `'3Y'`, `'5Y'`
- **Range affects bar resolution**: `D`+`12M` = daily bars, `W`+`60M` = weekly bars. Wrong combo = wrong timeframe
- **Sequential chart loading**: Load daily first → wait for iframe load → then weekly. MutationObserver pattern.
